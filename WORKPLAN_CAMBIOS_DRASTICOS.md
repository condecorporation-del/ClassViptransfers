#### 8A — Resend (correo oficial)
1. Entra a [resend.com/domains](https://resend.com/domains)
2. Agrega el dominio: `classviptransfers.com`
3. Resend te dará registros DNS (TXT, CNAME) → se los mandas al admin del dominio
4. Una vez verificado, cambia en Render:
   - `   
   - `COMPANY_BOOKINGS_EMAIL=armando@classviptransfers.com`
5. Prueba enviando email de test desde admin

#### 8B — PayPal Producción
1. El dueño (Armando) necesita cuenta PayPal Business en [paypal.com/business](https://paypal.com/business)
2. Dentro de PayPal Developer → Apps & Credentials → **Live** → Create App
3. Copiar `Client ID` y `Secret` del modo **Live**
4. En Render cambiar:
   - `PAYPAL_ENV=live`
   - `PAYPAL_CLIENT_ID=<el nuevo client id live>`
   - `PAYPAL_CLIENT_SECRET=<el nuevo secret live>`
5. Probar con una reserva real de $1 USD y luego reembolsar

#### 8C — Google Analytics
1. Crear cuenta en [analytics.google.com](https://analytics.google.com)
2. Agregar propiedad: `classviptransfers.com`
3. Copiar el Measurement ID (formato `G-XXXXXXXXXX`)
4. Yo agrego el script en `index.html` con ese ID
